# 自主制作課題 企画書（骨子）

## 1. テーマ名（仮）
自己成長・学習記録アプリ（＋目標管理機能）

## 2. 制作目的
日々の学習内容や気づきを記録し、自分の成長を可視化する。  
また、学習や習慣を継続するモチベーションを高めるための仕組みを取り入れる。  
→ 未経験エンジニアとして「継続的に学ぶ姿勢」をアピールできる内容に。

## 3. 想定ユーザー
- 学習を継続したい未経験エンジニア
- 自己成長を記録したい社会人・学生
- 習慣化を意識している人

## 4. アプリの概要
| 要素 | 内容 |
|------|------|
| 種類 | Webアプリ（Javaサーブレット＋HTML/CSS） |
| 使用技術 | Java / HTML / CSS / SQL（MySQL or SQLite） |
| 開発環境 | VSCode（Java拡張、Drawio拡張使用） |
| 実行時間 | 約60時間（5h×12日） |

## 5. 主な機能（予定）
| 機能区分 | 概要 | 実装予定 |
|-----------|------|-----------|
| 学習記録登録 | 日付・内容・気づきを入力・保存 | ○ |
| 記録一覧表示 | 過去の記録をテーブルで表示 | ○ |
| 目標登録 | 目標名と達成率を登録 | △（余力で） |
| 進捗表示 | 学習日数・達成率を集計 | △（簡易表示） |
| 削除機能 | 登録内容を削除 | ×（時間があれば） |

## 6. UI構想（仮）
- トップページ：学習記録一覧＋「新規登録」ボタン  
- 登録ページ：フォーム入力（日付／内容／気づき／目標進捗）  
- （余裕があれば）統計ページ：記録数・達成率の可視化
