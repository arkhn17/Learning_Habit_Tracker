# 自主制作課題 企画書（骨子）

## 1. テーマ名
Learning Habit Tracker（学習習慣トラッカー）

---

## 2. 制作目的
- 日々の学習内容や気づきを記録し、あとから振り返れるようにする  
- 学習の積み重ねを可視化し、習慣化の安定につなげる  

---

## 3. 想定ユーザー
- 学習習慣をつけたい人  
- 週〜月単位で学習を振り返りたい人  
- 記録や振り返りを通じて成長を把握したい人  
- 手軽に記録できるツールを求める人  

---

## 4. アプリ概要
| 要素 | 内容 |
|------|------|
| 種類 | Webアプリ |
| 使用技術 | JavaScript / Java / HTML / CSS |
| 開発環境 | VSCode / Draw.io / GitHub |

---

## 5. 主な機能（予定）
| 機能区分 | 内容 | 実装状況 |
|-----------|------|-----------|
| 学習記録登録 | 日付・内容・気づきを入力・保存 | ○（基本登録） |
| 学習記録編集 | 登録内容の更新・削除 | △（編集・削除機能など一部未実装） |
| 記録一覧表示 | 過去の記録をまとめて表示 | △（一覧作成中） |
| 目標設定・達成状況表示 | 目標登録、達成度の可視化 | △（トップページの簡易表示のみ） |
| 統計・振り返り | 日数・達成率などの集計表示 | ×（未実装） |
