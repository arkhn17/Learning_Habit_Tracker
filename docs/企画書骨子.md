# 自主制作課題 企画書（骨子・最新版）

## 1. テーマ名
Learning Habit Tracker（学習習慣トラッカー）

---

## 2. 制作目的
- 日々の学習内容や気づきを記録し、あとから振り返れるようにする  
- 学習の積み重ねを可視化し、習慣化の安定につなげる  
- 未経験エンジニアとして、「継続的に学ぶ姿勢」をアピールできる制作内容とする

---

## 3. 想定ユーザー
- 学習習慣をつけたいすべての人  
- 週〜月単位で学習を振り返りたい人  
- 記録／振り返りを通じて成長を把握したい人  
- 手軽に記録できるツールを求めている層  

---

## 4. アプリ概要
| 要素 | 内容 |
|------|------|
| 種類 | Webアプリ（HTML/CSS + JavaScript / Java） |
| 使用技術 | JavaScript / Java / HTML / CSS |
| 開発環境 | VSCode + Draw.io + GitHub |

---

## 5. 主な機能（予定）
| 機能区分 | 内容 | 実装状況 |
|-----------|------|-----------|
| 学習記録登録 | 日付・内容・気づきを入力・保存 | ○（基本登録） |
| 学習記録編集 | 登録内容の更新・削除 | △（編集・削除機能一部未実装） |
| 記録一覧表示 | 過去の記録をまとめて表示 | ○（一覧は作成中） |
| 目標設定・達成状況表示 | 目標登録、達成度の可視化 | △（トップページ簡易表示のみ） |
| 統計・振り返り | 日数・達成率などの集計表示 | ×（時間の都合上） |

---

## 6. UI構想（最新版）
### 6.1 コンセプト
日々の学習内容や気付きを簡単に記録し、あとから振り返れる Web アプリ。  
学習の積み重ねを見える化し、“習慣の安定”や“成長の把握”につなげる。

### 6.2 UI要件・スタイル方針
- シンプルで落ち着いた配色（薄紫／スカイブルーにグレー混ぜた柔らかい色合い）  
- テーマカラー変更に対応（配色設定画面）  
- 情報密度は低め、整然とした表示  
- 「要点だけ」表示 → 詳細はモーダル表示  
- 操作ステップを最小化  
- 画面上部に固定ヘッダー（アプリ名表示）、下部にナビゲーション  

---

### 6.3 ページ構成
#### 0. 新規登録／ログインページ
- アプリ名・コンセプト表示  
- メールアドレス／パスワード入力  
- 「ログイン」「新規登録」ボタン  
- 「パスワードを忘れた場合」リンク  

#### 1. トップページ
- 学習状況表示（バーグラフ + ％）  
- 主要アクションボタン  
  1. 学習を記録 → 記録ページ  
  2. 振り返り → カレンダーページ  
  3. 目標を確認 → 目標ページ  

#### 2. カレンダー／振り返りページ
- 月間カレンダー表示  
- 日付クリックでその日の記録リスト展開  
- 記録クリックで編集モーダル（保存・削除）  
- 右下「＋」ボタンで新規登録モーダル  
- 前月／翌月切替  
- 月合計学習時間表示  

#### 3. 目標設定ページ
- 目標リスト一覧（複数可）、進捗バー表示  
- 各目標クリックで編集モーダル（保存・削除）  
- 右下「＋」ボタンで新規目標モーダル  
- 週／月切替タブ  

#### 4. 設定ページ
- プロフィール編集（ユーザー名／メール／パスワード）  
- 配色テーマ変更  
- データ管理（CSVエクスポート、初期化）  
- その他（利用規約、プライバシーポリシー、ログアウト）  

---

### 6.4 全体共通要素
- ヘッダー固定＋ハンバーガーメニュー  
- 下部ナビゲーション（トップ／記録／目標／設定）  

---

## 7. 画面遷移図案
#### 0. 新規登録／ログインページ
- ログイン成功 → 1.トップページ  
- ログイン失敗 → 0.ログインページ（エラー表示）  
- 新規登録成功 → ログイン操作 → 成功でトップページ、失敗でログインページ  
- パスワード再設定 → 成功でログイン操作、失敗で再設定フォーム  

#### 1. トップページ
- 「学習を記録」 → 新規登録モーダル → 保存成功でトップページ反映  
- 「振り返り」 → 2.カレンダーページ  
- 「目標を確認」 → 3.目標設定ページ  
- ハンバーガーメニューから各ページ遷移  

#### 2. カレンダー／振り返りページ
- 日付タップ → 記録リスト展開 → 編集モーダル  
- 右下「＋」 → 新規登録モーダル  
- ハンバーガーメニューから各ページ遷移  

#### 3. 目標設定ページ
- 目標カードクリック → 編集モーダル  
- 右下「＋」 → 新規目標モーダル  
- 週／月切替タブ  
- ハンバーガーメニューから各ページ遷移  

#### 4. 設定ページ
- プロフィール編集モーダル  
- 配色テーマ変更  
- データ管理（CSVエクスポート・初期化）  
- その他（利用規約・プライバシーポリシー・ログアウト）  
- ハンバーガーメニューから各ページ遷移
