# Learning Habit Tracker

## 概要
日々の学習内容や気付きを記録し、あとから振り返れるようにするためのWebアプリです。  
職業訓練校での自主制作課題として開発しています。  
学習の積み重ねを見える化し、学習習慣の安定を目的とします。  

---

## 使用技術
- **プログラミング言語**：JavaScript / Java  
- **マークアップ言語・スタイルシート**：HTML / CSS  
- **ツール**：VSCode / Draw.io / GitHub  

---

## 機能（予定）
- 学習記録の登録・編集（新規／更新／削除）  
- 過去の記録をまとめて確認できる一覧ページ（アーカイブ）  
- 記録一覧の簡易表示（トップページ）  
- 目標設定と達成状況の確認（達成度の見える化）
